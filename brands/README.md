# Brand Portfolio Governance

## Purpose
This folder governs all HamMediaLabs brands.

## Philosophy
Brands are experiments until proven otherwise.

## Required per brand
- Brand profile
- KPI baseline
- Monetization hypothesis
- Kill / hold / scale thresholds
- Account registry
- Provider map

## Default launch count
3 brands maximum until operating rhythm is proven.

## Folder suggestion
```text
brands/
  templates/
  brand-a/
  brand-b/
  brand-c/
```

## Rule
No emotional attachment. Data decides.
