# Architecture Blueprint

## Core design
### Hydra Core (HQ)
Central strategic layer responsible for:
- Governance
- Scheduling
- Claude Code orchestration
- Playwright orchestration
- KPI aggregation
- Decision engine
- Global memory

## Hydra Heads (Brand Nodes)
Each brand node handles:
- CMS
- Brand voice
- Content generation
- SEO
- Social publishing
- Monetization
- Local analytics

## Preferred deployment model
### HQ:
One control plane

### Brands:
Multiple lightweight deployment nodes

## Data layers
### Global:
Cross-brand intelligence

### Local:
Brand-specific memory and voice

## Key principle
Centralized intelligence, distributed execution.
