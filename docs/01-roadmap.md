# 90-Day Roadmap

## Phase 1 — Foundation, days 1-14
- Complete documentation baseline
- Confirm project name and governance
- Define account matrix
- Define secrets model
- Select control plane hosting approach
- Create Claude Code implementation brief

## Phase 2 — Core build, days 15-30
- Build repo skeleton
- Build Playwright onboarding harness
- Build provider registry templates
- Build dashboard data model
- Build brand profile templates
- Build content QA workflow

## Phase 3 — Pilot brands, days 31-60
- Launch 3 controlled brand experiments
- Publish only through approved workflow
- Track traffic, engagement, cost, and conversion
- Review weekly

## Phase 4 — Decide, days 61-90
- Kill weak brands
- Hold uncertain brands
- Scale winning brand
- Convert learnings into reusable playbooks

## Success target
By day 90, HamMediaLabs should have one reusable operating system, three tested brand experiments, and a clear decision on what to scale next.
